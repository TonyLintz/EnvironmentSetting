# vscode 擴充功能

- MagicPython
- Python
- TODO Highlight
- Jupyter
- ESLint
- GitLens
- vscode-pdf
- Mithril Emmet: 前端語法快捷
- Live Server: 網頁即時顯示
- Beautify: 前端程式排版
- Python Test Explorer for Visual Studio Code: vsocde 執行測試

# 安装 pip

sudo apt-get install python-pip

$ sudo pip install virtualenv
